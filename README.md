Notes app made with Node js

## Build

```
npm run build
```

## Add note

```
node dist/app.js add --title="t" --body="b"
```

## Renove note note

```
node dist/app.js remove --title="t"
```

## List notes
```
node dist/app.js list
```

## Read notes
```
node dist/app.js read --title="t" --body="b"
```
